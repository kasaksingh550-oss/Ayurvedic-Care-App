# 🌿 Ayurvedic Care App

## 📱 Overview
The Ayurvedic Care App is a personalized Android application designed to provide users with Ayurvedic health solutions, remedies, and wellness guidance. It includes features like login/register, health history tracking, and a clean modern UI.

---

## ✨ Features
- 🔐 User Login & Registration system  
- 🏠 Home dashboard with Ayurvedic content  
- 📊 Health history tracking  
- 🧾 SQLite database integration  
- 🎨 Beautiful UI with custom backgrounds and animations  
- 🌿 Ayurveda-based health suggestions  

---

## 🛠️ Tech Stack
- Java (Android)
- XML (UI Design)
- SQLite (Local Database)
- Android Studio
